# Saltprint

## Setup

```sh
npm install
npm --prefix studio install
cp .env.example .env
cp studio/.env.example studio/.env
```

## Astro

```sh
npm run dev
```

## Sanity Studio

```sh
npm run studio:dev
```

## Check

```sh
npm run check
npm run studio:check
```

## Build

```sh
npm run build
npm run studio:build
```
